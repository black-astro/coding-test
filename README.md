프로그래밍을 처음 시작하는 사람도 **기초 문법부터 알고리즘까지 단계적으로 실력을 올릴 수 있도록** 만든
코딩 테스트 연습용 데스크톱 프로그램입니다. 난이도별 문제 풀이, 유형별 학습, 언어 문법 강의,
모의고사, 그리고 **실행 시간·메모리까지 측정하는 자동 채점**을 하나의 앱(PySide6, 다크 테마)에서 제공합니다.

## `$ 1. 개요`

| 항목 | 내용 |
|------|------|
| **목적** | 입문자가 문법 → 자료구조 → 알고리즘으로 단계적으로 연습하며 실력을 쌓는 것 |
| **형태** | 데스크톱 GUI(Windows) — 별도로 CLI 버전도 제공 |
| **지원 언어** | Python · Java · C++ · JavaScript (문제 풀이/채점), CSS · SCSS (웹 문법 학습) |
| **채점** | 정답 여부 + 케이스별 실행 시간(ms) + 최대 메모리 측정 |
| **수록 콘텐츠** | 문제 276(중복 정리·티어 재조정 반영) · 문법 강의 155 · 영어 단어 5,190 |

> 이 프로그램의 목표는 단순히 문제를 많이 푸는 것이 아니라, **초보자가 "왜 이렇게 푸는지"를 이해하고
> 한 단계씩 올라가도록** 돕는 데 있습니다. 그래서 모든 문제에 단계별 힌트와 정답 풀이가 붙어 있습니다.

## `$ 3. 설치 및 실행`

### 무설치판으로 시작하기 (일반 사용자 — 이 방법을 권장합니다)

설치 과정 없이 압축만 풀면 바로 쓸 수 있는 **무설치판**을 배포합니다. Python 도 함께 들어 있어
따로 설치할 것이 없습니다.

1. GitHub **[Releases](https://github.com/black-astro/coding-test/releases)** 페이지로 이동합니다.
2. 최신 버전의 **`codeT-v1.2.1-win64.zip`** 파일을 내려받습니다.
3. 압축을 풉니다(원하는 폴더 어디든 가능).
4. 풀린 폴더 안의 **`codeT.exe`** 를 더블클릭하면 실행됩니다.

> 별도 설치·관리자 권한이 필요 없습니다. 폴더를 통째로 옮기거나 USB 에 담아 다른 PC 에서 바로 실행해도 됩니다.
> 삭제할 때도 폴더만 지우면 됩니다.
>
> 기본 제공되는 Python 으로 Python 문제는 바로 채점됩니다. **Java · C++ 로도 풀고 싶다면** JDK / g++ 가
> 추가로 필요합니다(아래 6번 또는 앱 안 **가이드** 참고). 설치하지 않으면 해당 언어 채점만 비활성화될 뿐
> 프로그램은 정상 동작합니다.

### 소스로 실행 (개발자)

위 무설치판이면 충분하며, 아래는 코드를 직접 수정·기여하려는 경우에만 필요합니다.

요구 사항

| 구분 | 필요 조건 | 없을 때 |
|------|-----------|---------|
| 필수 | Python 3.10+ | — |
| 선택 | JDK 17+ | Java 채점만 비활성화 |
| 선택 | g++ (MinGW-w64) | C++ 채점만 비활성화 |
| 선택 | Node.js | JavaScript 채점만 비활성화 |
| 자동 | libsass(파이썬 패키지) | SCSS 컴파일 |

```bash
pip install -r requirements.txt      # PySide6, qtawesome, libsass
python gui.py                         # GUI 실행
python main.py                        # CLI 버전(선택)
```

> 컴파일러/런타임이 설치되지 않은 언어는 해당 언어의 채점만 자동으로 비활성화되며, 프로그램 자체는 정상 동작합니다.

## `$ 5. 사용 방법`

> 아래 설명은 **무설치판(`codeT.exe`)** 을 실행한 화면 기준입니다. Releases 에서 zip 을 받아 압축을 풀고
> `codeT.exe` 를 실행한 상태에서 그대로 따라 하면 됩니다.

### 문제 풀이

1. 사이드바에서 문제를 선택하면 가운데에 문제·예제·제한(시간/메모리)이 표시됩니다.
2. 상단에서 풀 **언어**(Python/Java/C++/JS)를 고릅니다.
3. 오른쪽 **console** 에디터에 코드를 작성합니다.
4. **Run(F5)** 으로 직접 입력을 넣어 출력을 확인하거나, **제출(Ctrl+Enter)** 로 채점합니다.
   - 결과는 아래 **terminal** 에 케이스별 정답/오답 + 실행 시간 + 최대 메모리로 표시됩니다.
   - 모든 케이스를 통과하면 진행도가 저장되고 단계 게이지에 반영됩니다.

### 힌트

- 문제 영역에서 **우클릭** → 1 · 2 · 3 · 마지막 단계 순서로 힌트를 엽니다.
- 단계가 올라갈수록 자세해지며, 마지막 단계는 **정답 코드와 풀이 설명**까지 보여 줍니다.
- 막혔을 때만 단계적으로 열어 보며 스스로 푸는 연습을 권장합니다.

### 집중(하이드) 모드

- 상단 **집중** 버튼 또는 `Ctrl+H` 로 켭니다.
- 사이드바와 문제 설명을 감추고 **코드 에디터 + 터미널 + 문제 선택**만 남깁니다.
- 문제 내용을 이미 파악한 뒤, 코드 작성과 결과 확인에만 집중할 때 사용합니다.

### 시스템 트레이

- 창의 닫기(X)를 누르면 **종료 / 트레이로 보내기** 중 선택할 수 있습니다(설정에서 기본값 지정 가능).
- 트레이 아이콘을 클릭하면 다시 열리고, 우클릭 메뉴에서 열기·집중 모드·종료를 선택할 수 있습니다.

### 모의고사

- 시험 섹션에서 프리셋을 고르면 여러 문제가 **무작위로 출제**되고 제한 시간이 시작됩니다.
- 시간 안에 문제를 풀어 제출하고, 시간이 끝나면 자동 채점되어 합격/불합격이 표시됩니다.

## `$ 6. 채점 방식`

| 표시 | 의미 |
|------|------|
| PASS | 정답 |
| FAIL | 오답(출력 불일치) |
| TIMEOUT | 시간 초과 |
| MEM | 메모리 초과 |
| ERROR | 런타임 에러 |
| BUILD ERR | 컴파일 에러 |

- **시간·메모리 제한은 문제마다 개별 적용**됩니다. 문제별 `time_limit_ms` / `memory_limit_mb` 가 있으며,
  지정되지 않은 경우 난이도 기본값을 사용합니다.
- 난이도별 기본 제한: 입문 1000ms/128MB · 초급 1500ms/256MB · 중급 2000ms/256MB · 상급 3000ms/512MB.
- 표준 입출력형 문제는 Python · Java · C++ · JavaScript 모두 채점하며, 함수 구현형 문제는 Python 으로 채점합니다.

### 환경 설치 요약 (JDK / g++)

앱의 **가이드** 섹션에 단계별 안내가 있습니다.

- **Java(JDK):** https://adoptium.net 에서 JDK 설치(또는 `winget install EclipseAdoptium.Temurin.21.JDK`).
  설치 후 `javac -version` 이 인식되면 자동 적용됩니다.
- **C++(g++):** https://winlibs.com 의 MinGW-w64 를 풀어 `...\mingw64\bin` 을 PATH 에 추가.
  `g++ --version` 이 인식되면 자동 적용됩니다.

## `$ 8. 빌드 및 검증`

```bash
pip install pyinstaller
python build.py        # dist/codeT/codeT.exe 생성
```

자세한 toolchain 동봉 방법은 **BUILD.md** 를 참고하세요.

### 검증

```bash

python tools/coverage.py        # 난이도/유형 커버리지 출력
python tools/verify_batch.py problems/batches/bronze_a.py   # 배치 파일 1개 정밀 검증
python tools/verify_lessons.py lessons/content/py_mid.py    # 레슨 예시 코드 실행 검증
```
